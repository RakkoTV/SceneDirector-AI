# SceneDirector-AI 中文文档

SceneDirector-AI 是一个具有 AI 预测功能的智能场景切换插件。

## 功能

- **多种切换模式**
  - 窗口焦点检测
  - 音频电平触发
  - 时间表
  - AI 预测

- **WebSocket API**
  - 外部控制
  - JSON 消息格式

## 安装

1. 从发布页下载 `SceneDirectorAI.dll`
2. 复制到 `obs-plugins/64bit/`
3. 重启 OBS Studio

## 配置

| 设置 | 描述 | 默认值 |
|------|------|--------|
| 切换模式 | 模式选择 | 窗口焦点 |
| 启用预测 | 使用 AI 预测 | 开 |
| WebSocket | 端口 | 8080 |
